## рҹ“„ Sobre este archivo HTML

El sitio web de RabGamesStudio estГЎ construido como un **Гәnico archivo HTML autocontenido** (`rabgamesstudio_v3.html`). No requiere servidor, base de datos, ni dependencias instaladas localmente вҖ” solo un navegador web moderno.

### TecnologГӯas usadas

| TecnologГӯa | Uso |
|---|---|
| HTML5 | Estructura semГЎntica |
| CSS3 (puro) | Estilos, animaciones, CRT effect, grid |
| JavaScript (vanilla) | Interactividad, slider, modal, FAQ |
| Google Fonts CDN | `Press Start 2P` В· `Share Tech Mono` В· `Rajdhani` |
| ImГЎgenes base64 | Screenshots de His Destiny y Talk to the Hand embebidos |
| ImГЎgenes CDN (GameJolt) | Thumbnails, avatares y banners cargados por URL |

### Requisitos para funcionar

- Navegador moderno (Chrome, Firefox, Edge, Safari 2020+)
- ConexiГіn a internet para: Google Fonts, imГЎgenes CDN de GameJolt
- Sin conexiГіn: funciona parcialmente (texto, CRT, slider con placeholder, interactividad JS)
